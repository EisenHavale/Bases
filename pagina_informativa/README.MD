# Primer proyecto. Pagina informativa

## Objectivos

1. Establecer una estructura para el manejo de html: 
```
<Header>
<Main>
<Footer>
```
2. Creacion de textos detallados y facil de seguir con el uso de etiquetas como:
```
<span>
<small>
<strong>
<a>
<br>
<hr>
```
3. Uso de enlaces e iconos
```
<a>
<img> / <svg>
```
